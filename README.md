# 프로젝트 시작하기

- expo 설치
  `npm install -g expo-cli`

- 프로젝트 시작
  `expo init GuessNumber`
  - blank (TypesScript) 선택

- android studio 설치 
  - 실행 후 More Actions > VDM >  Pixel 5 추천 

- Xcode 설치 
  - Finder > xcode > contents > developer >  application > simulator

- 프로젝트 실행 
  `npm start`
  - I > ios
  - A > android 실행 가능

- QR 코드 > 휴대폰에서 Expo go 설치 하고 찍으면 휴대폰에서 실행 가능
